Tool made by flamespill for carzonedb.github.io
It is recommended to download the HTML and run it locally on your PC.
When run online (e.g. via GitHub), requests may go through shared servers and hit global rate limits. Running locally uses your own IP, making downloads faster and more reliable.


• Batch
How many cars are requested per API call.
→ 100 = best balance (fast + safe)

• Resolution
→ 150x150 = fastest, low quality
→ 420x420 = best overall (recommended)
→ 700x700 = highest possible

• Skip invalid
Skips broken, empty, or not-yet-generated thumbnails.

• ZIP filename
Saved as:
CarZone_images_YYYY-MM-DDTHH-MM-SS.zip
Example:
CarZone_images_2026-03-18T12-34-56.zip
→ Includes the exact date & time of download

Speed is auto-adjusted to avoid rate limits.
